Toda web tiene 3 conceptos básicos:
HTML: es la estructura, la información de la pagina web
CSS: es el maquillaje para HTML, para que la informacion que está en HTML se vea bonito
JavaScript: el que le da vida a la web.
Siempre nos han enseñado que estos tres conceptos deben de ir separados,  pero en React
introducimos estos 3 conceptos en un mismo componente.

¿QUÉ ES DOM? - El DOM es la estructura de objetos que genera el navegador cuando se carga un documento y 
se puede alterar mediante Javascript para cambiar dinámicamente los contenidos y aspecto de la página.
DOM virtual es que es básicamente una representación del DOM pero en memoria.
